OCP - Loja de Motociclismo

Principio do "Aberto-Fechado". Aberto para novas inclusões, porém fechado para alterações.

O objeto "Moto" é uma classe que será herdada/extendida para todos os veículos implementados, caso um modelo com particularidades surja uma nova classe que herda moto será criada, evitando assim alterações diretamente no objeto moto que tem atributos mais genéricos.
